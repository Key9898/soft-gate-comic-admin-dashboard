# Folder Map & Architecture

```
src/
├── components/   # Standalone global primitives
├── config/       # Constants & configurations
├── data/         # Mock data seeding
├── features/     # Organized logically by platform features
├── layouts/      # Global layouts (AdminLayout)
├── lib/          # Custom utility libraries (i18n, formatting)
└── types/        # TypeScript declarations
```
